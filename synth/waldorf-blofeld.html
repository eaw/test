Sending sounds from Vim to the Blofeld

Execute the following (adjust hw:2,0,0 to match the correct amidi port):

  $ cd ~/Downloads/waldorf-blofeld/synth.stromeko.net/
  $ find -name *.syx | grep -i singles | sed 's:^[.]/::' | sort | view -
      Vim:
      :map <f12> :sil .w ! xargs -i bash -c 'amidi -p hw:2,0,0 -s "{}"'<Enter>
        or
      :map <f12> :sil .w ! shellquote \| xargs -i bash -c 'amidi -p hw:2,0,0 -s "{}"'<Enter>

Now pressing F12 in Vim will send the sound(s) in the file on the current line to the Blofeld (except the ones with single quotes in their name -- I don't know how to deal with them yet).
